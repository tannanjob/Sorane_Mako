使用に当たって
-----------
フォントをインストールして頂くと、ワープロ、グラフィックソフトなどの

概要
-----------

ラノベPOPv2は、以下のフォント
にオリジナルの文字を加え加工したフォントです。

* M+（エムプラス）フォント

ひらがな、カタカナをオリジナルで制作し、残りの部分をM+FONTで補いました。
M+FONTの派生プログラムにあたります。
